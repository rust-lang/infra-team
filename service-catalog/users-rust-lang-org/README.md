# users.rust-lang.org

[users.rust-lang.org] is the Rust users forum for community support and discussion.
It is deployed on Discourse.

## Discourse

[Discourse] is an open source discussion platform for forums and community
support.

## Support

For Discourse support tickets related to `users.rust-lang.org`, you can send a PM
to `team` directly in Discourse or email `team@discourse.org`.

[discourse]: https://www.discourse.org/
[users.rust-lang.org]: https://users.rust-lang.org/
